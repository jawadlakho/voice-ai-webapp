# 🎙️ Voice AI Studio

A modern, feature-rich Voice AI web application that converts Text-to-Speech and Speech-to-Text using the browser's native Web Speech API.

## ✨ Features

### Text → Speech
- 🗣️ Multiple language support (13+ languages)
- 🎭 Voice selection (all system voices per language)
- ⚡ Speed control (0.5x to 2.0x)
- 📋 Copy text to clipboard
- 💾 Download transcript as .txt file

### Speech → Text
- 🎤 Real-time microphone recording
- 🌍 Language selection for recognition
- 📄 Continuous transcription with interim results
- 📋 Copy recognized text
- 💾 Download transcription as .txt file

### UI/UX
- 🌙 Clean dark mode design
- 💎 Glassmorphism effects
- 📱 Fully responsive (mobile friendly)
- ⚡ Smooth animations and transitions
- 🎯 Intuitive controls

## 🚀 How to Run

### Method 1: Direct (No installation)
1. Download all files maintaining the folder structure
2. Double-click `index.html` to open in your browser
3. Allow microphone access when prompted

### Method 2: Using VS Code Live Server (Recommended)
1. Open the project folder in VS Code
2. Install "Live Server" extension
3. Right-click `index.html` → "Open with Live Server"

### Method 3: Python HTTP Server
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
